# Design Style: Blink

앱의 비주얼 시스템. 기존 Newsprint(흑백 신문) 스타일을 대체한다. 깔끔한 카드 기반 모바일 UI — iOS 감성의 밝은 배경, 둥근 카드, 파란 액센트.

## 1. 철학

- **밝고 가벼움**: 순백이 아닌 아주 옅은 회색 배경 위에 흰 카드가 떠 있는 구조.
- **부드러운 기하학**: 큰 라운드(14–18px), 얇은 보더, 은은한 그림자. 신문식 각진 보더/제로 라운드는 쓰지 않는다.
- **파란 액센트, 절제된 사용**: 링크·활성 상태·강조에만 파란색. 나머지는 무채색.
- **정보 카드**: 서클 하나 = 카드 하나. 배지(부스번호) · 제목 · 장르 · 태그 · 링크칩 · 방문 체크로 구성.

## 2. 디자인 토큰 (`src/index.css`의 `@theme`)

색상
- `--color-bg: #f4f5f7` — 앱 배경
- `--color-card: #ffffff` — 카드
- `--color-ink: #17181c` — 기본 텍스트
- `--color-muted: #6b7280` — 본문 보조
- `--color-faint: #9aa0aa` — 메타/라벨
- `--color-line: #ecedf0` — 보더
- `--color-chip: #f1f3f6` — 태그/링크 칩 배경
- `--color-accent: #3e5bff` — 액센트(블루). 소프트 틴트는 `bg-accent/10`, 보더는 `border-accent/30`.

배지 팔레트(카드 앞 아이콘): `#3e5bff #7c5cff #f43f72 #0ea5a5 #f59e0b #6366f1 #22a559` — 서클 순서대로 배정.

타이포
- 단일 서체: **Pretendard**(`--font-sans`). 별도 serif/mono 없음.
- 굵기 대비로 위계: 화면 타이틀 22px/800, 카드 제목 16.5px/800, 본문 13.5–15px, 메타 11–12.5px/700.
- 자간: 큰 타이틀은 `-tracking-[0.02em]`, 라벨은 `tracking-[0.05em]`.

라운드 / 보더 / 그림자
- 라운드: 카드 18px, 검색·버튼 14px, 칩 999px(pill), 배지 10px.
- 보더: 1px `--color-line`. 강조(걸밴크 전문) 카드는 `ring-[1.6px] ring-accent/40 border-transparent`.
- 그림자: 카드 `0 1px 2px rgba(20,22,30,.04)`, 프레임 `0 30px 70px -20px rgba(20,22,30,.34)`. 하드/오프셋 그림자 금지.

## 3. 컴포넌트

**칩(필터)**
- 상태칩(전체/체크함/안 본 것): 활성 `bg-ink text-white`, 비활성 `bg-card text-[#7b818c] border-line`. h-34, pill.
- 장르칩(전체 장르/걸즈밴드크라이/뱅드림): 활성 `bg-accent/10 text-accent border-accent/30`. 가로 스크롤.

**검색바**: 흰 배경 pill, 좌측 돋보기, 우측 지도핀(전체 배치도 링크). h-48, 라운드 14px.

**카드**: 앞에 부스번호 배지(연속 부스는 앞쪽만, 예 BE01·BE02→BE01) → 카테고리(부스 대문자) + 걸밴크 전문 pill → 제목 → 장르 → note(옅은 회색 박스) → 장르 태그 → 링크칩(배치도=액센트 틴트, X/Insta/샵/윗치폼=회색). 우측 상단 원형 방문 체크.
- 체크됨: `bg-[#f3f5f8]`, 투명도↓, 제목 취소선.

**상세**: 뒤로가기 헤더 → 큰 배지+타이틀 → (전문 pill) → 장르 → 태그 → MEMO 카드 → 링크 버튼(풀폭) → 하단 "방문 체크"(미체크=검정, 체크=액센트).

**하단 네비 없음.** 배치도는 검색바 핀으로 접근.

## 4. 레이아웃

- 모바일 우선. 콘텐츠 폭 432px 프레임을 중앙 정렬(반응형: 데스크톱에선 중앙에 배치).
- 세로 카드 리스트, `gap-3`, 좌우 패딩 20px.
- 헤더+검색+상태칩은 스크롤 시 상단 고정(sticky).

## 5. 상태

- 방문 체크는 `localStorage["gbc-seoko-2026-07-checks"]`에 저장(기존 키 유지).
- 필터(상태/장르)·검색은 클라이언트 상태. 카드 탭 → 상세, 뒤로가기 → 목록.

## 6. 모션

- `transition-colors` 위주, 빠르고 담백하게. 바운스/과한 이징 지양.
